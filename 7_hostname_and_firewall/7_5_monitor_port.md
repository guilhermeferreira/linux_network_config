# Exploring Sockets

| Package | Tool |
|---------|------|
| `net-tools` | `netstat` |
| `iproute` | `ss` |

## Options

| Option | Displays |
|--------|--------|
| `-4` | Active IPv4 sockets |
| `-t` | Active TCP sockets |
| `-u` | Active UDP sockets |
| `-l` | Listening sockets |
| `-p` | Process PID responsible for the socket |
